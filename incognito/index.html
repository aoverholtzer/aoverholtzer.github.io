Incognito is a private web browser you can pull up anywhere.

From any app: long-press a link or select text, choose “Share”, and then choose the Incognito action extension. Boom: instant web browser!

Incognito has no ads, no third-party trackers, and does not send or re-route your web searches anywhere.
