Description

Switch keyboard languages with ease! Switcheroo is a menubar app that shows your input sources with colorful flags — or with a custom image of your choosing! Use the menu to switch, add, or remove input sources with just a few clicks. It’s the perfect app for polyglots and vexillologists alike.

  • See flag icon for your current keyboard language / input source.
  • Menu lists all enabled input sources. Click to switch inputs.
  • Click “Edit” to add or remove inputs, or to edit icons. Choose from flag, emoji, or default icons, or select any image from your computer.
  • Emoji picker available too!
  • Quick access to Keyboard Preferences, for advanced options.

To configure fn hotkey, go to System Preferences > Keyboard > Input Sources.

If you find any languages or icons missing, please contact me at adam@overdesigned.net or @aoverholtzer on Twitter!
